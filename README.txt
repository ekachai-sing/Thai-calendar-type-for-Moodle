The Thai calendar is a solar calendar which use Buddish year, Thai month name, Thai date name, the official calendar in Thailand.

The calendar type system was introduced in 2.6, so this will not work in any Moodle versions earlier than this.

To install this calendar type do either of the folllowing steps.

	1. Download and extract the zip, rename the folder to thai and place this folder in the calendar/type folder in your Moodle directory.
	2. Visit http://YOURSITEMAINURL/admin/index.php and update database.

Credit : This calendar's source code get ideas from Jalali calendar.



------------------------------------------ ภาษาไทย -----------------------------------------
ปฏิทินไทย เป็นปฏิทินตามสุริยคติ ใช้ปี พ.ศ. เดือนชื่อไทย วันชื่อไทย ซึ่งเป็นปฏิทินใช้ตามราชการไทย

ปฏิทินนี้เริ่มต้นใช้ได้กับ Moodle 2.6 ขึ้นไปเท่านั้น เวอร์ชั่นก่อนหน้าจะไม่สามารถใช้งานได้

การติดตั้งให้ปฏิบัติดังนี้

	1. ดาวน์โหลดและแตกไฟล์ แล้วสร้างโฟลเดอร์ชื่อ thai ไว้ภายใต้โฟลเดอร์ calendar/type ของ Moodle
	2. ไปที่  http://YOURSITEMAINURL/admin/index.php  จากนั้นทำการอัพเดทระบบ

เครดิต : โค้ดของปฏิทินนี้ ได้แนวคิดมาจากปฏิทิน Jalali
